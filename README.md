<h1>WatchUniversityScheduleManager — Расписание на вашем запястье</h1>
Приложение для Samsung Galaxy Watch 6 Classic, позволяющее студентам просматривать расписание своей группы прямо на умных часах.
Интеграция с университетским API, минималистичный интерфейс, оптимизированный под маленький экран. 
<h2>Основные возможности</h2>
<ul>
  <li>Получение списка всех групп из университетского API</li>
  <li>Выбор нужной группы из списка</li>
  <li>Просмотр расписания выбранной группы (дни недели, время, предметы, аудитории)</li>
  <li>Минималистичный UI, адаптированный под круглый экран Galaxy Watch 6 Classic</li>
  <li>Простая интеграция с внешним API через HTTP-запросы</li>
</ul>
